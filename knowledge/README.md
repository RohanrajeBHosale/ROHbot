# ROHbot Knowledge Base (Public)

This folder contains the **public knowledge base** used by ROHbot for retrieval (RAG).

## What’s inside
- Sanitized project summaries
- Public links (portfolio, GitHub, writeups)
- Cleaned text snapshots of public pages

## What’s NOT inside
- Any private documents
- Personal identifiers (address, phone, student ID, etc.)
- Secrets or API keys

## Update policy
This KB is periodically refreshed from public sources.
